AI-Based Cancer Detection Site

  • Tech Stack: Flask, Python, JavaScript, HTML, CSS, Replit, TensorFlow, NumPy, OpenCV, Matplotlib
  • Github URL: https://github.com/KoralK5/CWSF2021
  • Youtube Link: https://www.youtube.com/watch?v=UF0MKbz8_-U
  • Project Board: https://projectboard.world/ysc/project/cancer-detection-with-optimized-neural-networks

A Canada-Wide Science Fair project consisting of a Flask website which contains 3 TensorFlow-optimized cancer detection neural networks. Users can test themselves for cancer by submitting their medical scans to the website.

  • Invented a new neural network optimizer.
  • Created multiple different neural network modules from scratch using NumPy
  • Created a Python testing environment using Matplotlib to analyse neural network optimizers.
  • Formatted medical datasets for AI training.
  • Built a prototype website using HTML/CSS/JS and Flask.
